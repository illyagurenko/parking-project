# Система Управления Автостоянкой

Веб-приложение для управления автостоянкой. 
Оно позволяет администратору вести учет клиентов, автомобилей, парковочных мест и управлять бронированиями.

## Стек технологий и База данных

Проект состоит из трех основных компонентов, упакованных в Docker-контейнеры:

1. **База данных**: **PostgreSQL 17**. 
2. **Бэкенд**: Java 21 + Spring Boot 3.5.14. 
3. **Фронтенд**: Vue.js 3 + TypeScript + PrimeVue. 

Подробнее о каждой части:
- [Документация Бэкенда](./backend/README.md)
- [Документация Фронтенда](./frontend/README.md)

## Основной функционал

* **Клиенты**: Добавление, редактирование, удаление, поиск по ФИО.
* **Автомобили**: Привязка автомобиля к клиенту, поиск по номеру машины.
* **Парковочные места**: Учет всех мест стоянки.
* **Бронирования**: Выделение места конкретной машине, отслеживание времени и статуса оплаты.
* **Поиск и Фильтрация**: Выпадающие списки с поиском, пагинация в таблицах.

## Быстрый запуск Docker

1. Откройте командную строку в корневой папке проекта.
2. Выполните команду:
   ```bash
   docker compose up -d --build
   ```
3. После завершения сборки приложение будет доступно по адресу: **http://localhost**
