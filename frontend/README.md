# Parking Frontend

Интерфейс для системы управления автостоянкой.

## Стек технологий

- **Vue.js 3** 
- **Vite** 
- **TypeScript**
- **Pinia** 
- **PrimeVue** 
- **Axios**

## Основной функционал

1. Управление клиентами и их автомобилями.
2. Управление парковочными местами.
3. Оформление бронирований с отслеживанием статуса оплаты.
4. Встроенная пагинация, постраничный вывод таблиц.
5. Динамические списки с поиском для удобного выбора (фильтрация).

## API и База данных

Фронтенд  общается с бэкендом через следующие REST API эндпоинты (базовый URL `/api`):
- `/clients` (GET, POST, PUT, DELETE)
- `/cars` (GET, POST, PUT, DELETE)
- `/parking-spaces` (GET, POST, PUT, DELETE)
- `/reservations` (GET, POST, PUT, DELETE)

Все данные (клиенты, машины, парковки, брони) хранятся в реляционной базе данных **PostgreSQL** на стороне сервера.

## Как запустить локально

1. Откройте терминал в папке `frontend`.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
